The Telegram Analytics Module¶

Design Notes¶

The Telgram Interface makes use of the Telegram Client API to grab message data and run analytics.

Implementation Documentation¶

A that extracts information about messages times.

get_all_message_date_times(username, limit=10)¶

Get dates and times for all messages exchanged between you and an entity.

Parameters:
  • username (str) – Username (or phone number) of entity to get chat history from.
  • limit (int) – The number of messages to extract information from.
Returns:

Array of dates and times that each messages was sent.
Return type:

[datetime.datetime]

 
>>> ca = TelegramAnalytics( api_id, api_hash )
>>> ca.get_all_message_date_times('+11234567891',limit=1)
>>> ca.get_all_message_date_times('quartz_husky',limit=1)

message_sentiment_vs_time(username, limit=10)¶

Get a graph of sentiment as a function of hour of the day.

Parameters:
  • username (str) – Username (or phone number) of entity to get chat history from.
  • limit (int) – The number of messages to extract information from.
Returns:

A tuple with the following format: (hour,sentiment,variance-in-sentiment)
Return type:

(int,float,float)

 
>>> from fnw_client import TelegramAnalytics
>>> import matplotlib.pyplot as plt
>>>
>>> plt.figure()
>>>
>>> ca = TelegramAnalytics( api_id, api_hash )
>>> sentiment_and_time = ca.message_sentiment_and_time('quartz_husky',limit=100)
>>> variances = [ x[2] for x in sentiment_and_time ]
>>>
>>> plt.bar(range(len(sentiment_and_time)), [val[1] for val in sentiment_and_time], align='center',yerr=variances)
>>> plt.xticks(range(len(sentiment_and_time)), [val[0] for val in sentiment_and_time])
>>> plt.xticks(rotation=70)
>>> plt.xlabel('Hour')
>>> plt.ylabel('Average Sentiment')
>>>
>>> plt.show()
